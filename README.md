# tz-tele-shop-bot

## 0.INIT
```bash
npm install
```
## 1. EXPORT IMPORTANT VARIABLES
  Linux:
   ```bash
   export BOT_TOKEN="telegram bot token"
   export TG_ADMIN_USERNAME="username for admin of bot"
   ```
  Windows: 
    the same, but use "set" instead "export"
## 2. BUILD
  ```bash
  npm run build
  ```
## 3. START
  ```bash
  npm run start
  ```
